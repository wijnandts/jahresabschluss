Durchschnittliche Kapitalbasis

Für eine korrekte Analyse wird der Gewinn auf das im Jahresverlauf durchschnittlich verfügbare Kapital bezogen.

ø Eigenkapital = (EK Anfang + EK Ende) / 2
ø Gesamtkapital = (GK Anfang + GK Ende) / 2

Die Rentabilitäten

Eigenkapitalrentabilität = (Gewinn / ø Eigenkapital) * 100
Gesamtkapitalrentabilität = ((Gewinn + Zinsen) / ø Gesamtkapital) * 100
Umsatzrentabilität = (Gewinn / Umsatzerlöse) * 100
